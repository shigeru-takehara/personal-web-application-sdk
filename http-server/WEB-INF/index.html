Personal Web Application (PWA) SDK

PWA SDK provides software components with which you can build your personal application within a web browser.

What is the "personal web application"? You start your application from a command file that will open the web page. When you close the web page, the application will be terminated so that you cannot view the web page any more. It works in your computer as you need. The size of an application should be less than 100MB.It's a really small application.

The default application framework is vue-js and primitive-ui css. You don't need to stick to these and you can use any tools you like.

Any IO operation between browser and server-side should be only "GET" method. Axios is used for "GET" call.

Please read PWA Tutorial first to build your application.

Download PWA-SDK from here.

* Currently, PWA works fully on only Windows environment.This is because some of functions used in PWA are specific to Windows OS. If it is executed on a different OS, it would throw an "MethodNotImplmented" Exception.

Installation

Downlad pwa.zip file from the above link, unzip to you "dev" directory, and execute startPWA.bat program. You will see this page in your browser.
