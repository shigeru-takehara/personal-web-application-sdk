PWA Tutorial: Introduction

The idea is "can we build a web application that just runs on my local computer?" Microsoft used to have Visual Basic, which allowed us to build an Windows application very easily. I wanted to have a tool like that for web applications because most of daily works is within the browser. It would be nice to have my own applications that help my daily work in the browser.

PWA includes a small web HTTP server that works with only "GET" method. You can develop web pages with any Javascripts framework. As described in the index page, Vue-JS and Primitive-UI CSS are used as default, and Axios is used for REST call.

  • "Hello World!" page is developed with HTML and Javascripts
  • "Synchronize Directories" page is developed with Vue-JS.

This tutorial provides how to develop HTML pages in the PWA way. Also, it will describe how to use Java objects developed for PWA, which helps to develop applications that support your daily work.

Each web page could become a single page application. For example, in the PWA directory you installed includes:

  • startHelloWorld.bat
  • startSyncDir.bat.
If you run them, each runs as a standalone application. it runs from another instanciated "Command Window". If you close the browser tab, the corresponding "Command Window" will be terminated. Just click the above two links. You can run the program from here.
