PWA Tutorial: PWA Directory

As you unzip PWA.zip file, you install the PWA directory. You will see following files:
File Name Description
HttpServer.js PWA Web HTTP Server
pwa.jar PWA Java Objects, which would help you to build your applications
  • startHelloWorld-tutorial.bat
  • startSyncDir-tutorial.bat
 PWA batch files described in the PWA Tutorial

You also see following directories:
Directory Name Description
WEB-INF You will store all HTML, JS, and CSS files used to build your applications. There are some default sub-directories, but you can add and/or modify as you like. "javadoc", "jsdoc", and "tutorial" directories are not needed for your applications. Feel free to delete what you don't use.
SERVER-SCRIPTS You will place server-side Javascripts in this directory. We will disucss it later.
MY-DATA This is the default data location for your applications.
TEMP This is the default temporary directory. You can use this directory for any purpose.
UTIL You can place any files to support your applications.
