PWA Tutorial: JAVA Objects

All PWA Java Classes are documented here. These classes would help you when you build applications that support your daily work. Some of your routine work depends on the result of some system, then you might execute some program to do, and then you might check something and do something else. If these routine work is manually executed, you could automate the routine work with PWA. The PWA Java classes could help you to build such automation application that could be executed from the browser.

Concept

Assume we run programs, which shows the result in its log file, on the console screen, does something that we could check. When a program is executed, CmdWinController object is used. This object could store the information displayed on the screen and could return the result of the program, but it cannot manage when the program finishes, which has to be managed by us with CmdCompletionChecker object, which provides several methods to check if the program finishes or not. To display processing status, CmdLogger is used.

CmdWinController Object

When this object is intantiated and used, we should set "tempFilePrefix" before calling executeCommand method. All processing tracking is done with temporary files in OS' temp directory. The temporary file is named with "tempFilePrefix" value as the file name prefix. For actual use, we should use pwa.commandWindow.initialize and pwa.commandWindow.close methods.

In PWA JavaScripts, this object is called "cmdWin".

CmdCompletionChecker Object

This object provides several methods that check if the command is finished or not. Use the methods whose name starts with "wait". For example, "waitForCommand", "waitByFileContent", etc. The processing will be waiting until specified condition is met. If the waiting condition is not the information on the console and in another external file, you should use the "wait" method that requires the external file name.

In PWA JavaScripts, this object is called "cmdWait".

CmdLogger Object

When your automation application starts, we may want to see processing status/result. Use this object for that purpose. For example, This is calling methods:

cmdLogger.log("##### Build ABC #####"); cmdLogger.logRun(command); cmdLogger.logDoneSuccess(command); cmdLogger.close(); The below is corresponding logging information in the log file. 2020-03-15 11:43:56.551 ##### Build ABC ##### 2020-03-15 11:43:59.693 Run: ABC build 2020-03-15 11:46:09.826 Done-Success: ABC build Lasted 0:2:10 2020-03-15 11:46:10.831 Overall Lasted 0:2:14

In PWA JavaScripts, this object is called "cmdLogger".
